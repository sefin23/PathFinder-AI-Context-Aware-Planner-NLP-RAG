# 🗺️ Pathfinder AI — Brand Identity Guide

This guide defines the visual language of the Pathfinder AI ecosystem.

## 🎨 Color Palette
The identity is built on a "Dark Forest" theme inspired by navigation and grounding.

| Color | Hex | Use Case |
| :--- | :--- | :--- |
| **Forest Deep** | `#06100a` | Global background, dark modes. |
| **Amber** | `#d47c3f` | Main action buttons, highlight icons. |
| **Emerald** | `#5c8c75` | Success states, timeline progress. |
| **Coral** | `#c65d4a` | Urgency, overdue tasks, danger zones. |
| **Sky Blue** | `#60a5fa` | Logo primary gradient, trust elements. |
| **Navy** | `#1e3a8a` | Logo secondary gradient, deep depth. |

## 📐 Typography
*   **Headings**: `Outfit` (700 Bold) — Modern, authoritative, accessible.
*   **Display/Italic**: `Playfair Display` — Warm, human, editorial feel.
*   **Body**: `Inter` or `DM Sans` — Clean, legible for complex data.
*   **Mono**: `JetBrains Mono` — Precision, technical markers.
